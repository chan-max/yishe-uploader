# SHEIN

## 支持场景

- `search`
- `product_detail`
- `shop_hot_products`

## 当前状态

- 已完成目录化和初版 DOM 规则整理。
- 历史测试中 SHEIN 比较容易触发验证码，当前先保证失败可识别、规则可维护。
- 2026-04-04 实测：搜索会跳到 `risk/challenge`，当前稳定识别为 `captcha`。

## 目标页面

- 搜索页：`https://us.shein.com/pdsearch/{keyword}/?page={page}`
- 详情页：商品详情页
- 热门商品页：商品流或列表页

## 选择器策略

- 商品链接优先使用 `-p-` 特征。
- 类目结果里保留 `-cat-` 作为补充识别。
- 标题和价格使用包含式 class 方案，避免绑定完整动态 class。

## 风险与验证码

- 当前最主要问题是验证码和访问限制。
- 若后续要提升稳定度，需要先解决平台风控，再继续优化选择器。

## 维护方式

1. 先判断当前页面是否已被验证码覆盖。
2. 如果页面正常，再确认 `-p-` 商品链接和商品名称节点。
3. 页面完全改版时，优先从商品链接模式重新定位卡片。
